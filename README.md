# RMRP Deals

Локальная витрина для легальных товаров и услуг RMRP.

## Запуск

Откройте `start-site.bat`, затем сайт будет доступен на `http://localhost:8080`.

Не используйте `file://.../index.html` для админки: каталог в таком режиме может показываться как демо, но регистрация и редактор работают только через сервер.
